家庭常备药物应满足以下几个原则：

1. **覆盖常见小病小痛（发热、感冒、腹泻、外伤等）**
2. **安全性高、说明清楚、使用简单**
3. **不滥用抗生素、激素类药物**

下面是按类别列出推荐常备药，**以成人健康人群为主**，若有老人、儿童、孕妇，需个别考虑。

---

## **一、解热镇痛类**

| 药名               | 说明                            |
| ---------------- | ----------------------------- |
| **对乙酰氨基酚（扑热息痛）** | 发烧、轻度头痛、牙痛、肌肉痛。肝功能不佳慎用。       |
| **布洛芬**          | 退烧+抗炎+镇痛，适合头痛、关节痛、月经痛。胃不好者慎用。 |

---

## **二、感冒咳嗽类**

| 药名               | 说明                  |
| ---------------- | ------------------- |
| **复方氨酚烷胺/氨麻美敏片** | 常见的感冒药，缓解鼻塞、流涕、发烧等。 |
| **右美沙芬**         | 镇咳，适合干咳。            |
| **氨溴索口服液/片**     | 化痰药，适合咳痰困难。         |
| **生理海盐水喷雾**      | 鼻塞时清洗鼻腔，儿童也能用。      |

---

## **三、胃肠类**

| 药名          | 说明                    |
| ----------- | --------------------- |
| **蒙脱石散**    | 止泻、吸附病毒和毒素，轻度肠炎有用。    |
| **口服补液盐**   | 腹泻或呕吐后补液、防脱水。         |
| **乳果糖/开塞露** | 便秘时使用，短期使用有效。         |
| **奥美拉唑**    | 胃酸过多、胃胀、反流。需注意不能长期滥用。 |
| **多潘立酮**    | 止吐助消化，胃动力不足时使用。       |

---

## **四、抗过敏类**

| 药名              | 说明                   |
| --------------- | -------------------- |
| **氯雷他定 / 西替利嗪** | 常用抗过敏药，适合皮疹、荨麻疹、鼻炎等。 |

---

## **五、外用类**

| 药名               | 说明                    |
| ---------------- | --------------------- |
| **碘伏**           | 伤口消毒，比酒精温和。           |
| **创可贴、纱布、绷带**    | 小创伤处理必备。              |
| **云南白药喷雾 / 三七粉** | 跌打损伤、瘀伤处理。            |
| **莫匹罗星软膏**       | 外伤感染处理（细菌性）。比红霉素软膏更强。 |
| **清凉油 / 风油精**    | 驱蚊、缓解头晕、轻微止痒。         |

---

## **六、其他常用**

| 药名             | 说明          |
| -------------- | ----------- |
| **体温计**        | 数字式或红外线式皆可。 |
| **血压计**        | 尤其家中有高血压老人。 |
| **止血带、小剪刀、镊子** | 简易外伤处理工具。   |
| **冰袋、暖宝宝贴**    | 用于冷敷/保暖等。   |
| **口罩、酒精棉片**    | 防护和清洁用途。    |

---

## **注意事项**

* 常备药**不应囤太多**，注意有效期，**每半年检查一次**。
* 所有药物都应**明确标注用途、剂量、禁忌**，避免误用。
* 有老人或儿童，请单独配儿童剂型和安全用药说明。
* 如家中有人**基础病（如高血压、糖尿病、心脏病）**，需个性化准备处方药。
