# 用阳极氧化法来烤蓝
## 工具总览  
![工具全家福](阳极氧化工具.jpg "打磨机,可乐(酸),等离子水,电源,保鲜膜,")  
##钛件操作流程  
1. 打磨  
    * 用800 1500 2000目的砂纸打磨成镜面,去掉原本氧化层 **  
2.  洗干净  
    - 用洗洁精洗干净,餐巾纸擦干。  
    - 注意别让镜面碰到手  
3. 阳极氧化  
    - 正极连接目标
    - 负极链接不锈钢或铅板
    - 注意阴阳两极不要碰撞,放入可乐中通电
4. 等离子水
    - 等待若干分钟后拿出来(根据电压颜色相关对照表决定)![相关对照表](电压色彩对照表.jpg)
    - 清水冲干净,放入等离子水,保鲜膜或塑料袋密封
    - 等待15分钟,让颜色更牢固
5. 吹干
    - 捞出来吹干  

**钛件的颜色和电压有关,所以根据电压大小就可以控制颜色**

##铝件操作流程
1. 脱脂,酸洗,去掉自然氧化膜
2. 放入稀硫酸溶液,温度20℃,铝材正极,负极接铅板,通电后会形成一层厚厚的有空氧化膜
3. 铝材洗干净,放入调配好的染料中,染料会渗透进孔洞中
4. 最后把铝材放入90℃的醋酸镍溶液中
