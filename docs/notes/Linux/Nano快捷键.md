# Nano 常用快捷键（简单但也要记）

> Nano 使用 `Ctrl` 作为前缀键，操作直观，适合快速编辑。

| 快捷键        | 功能说明         |
| ---------- | ------------ |
| `Ctrl + O` | 保存文件（Output） |
| `Ctrl + X` | 退出 Nano      |
| `Ctrl + G` | 显示帮助         |
| `Ctrl + K` | 剪切整行（删除行）    |
| `Ctrl + U` | 粘贴行          |
| `Ctrl + W` | 搜索文本         |
| `Ctrl + \` | 替换文本         |
| `Ctrl + C` | 显示光标位置（行列号）  |
| `Alt + 6`  | 复制当前行        |
| `Ctrl + _` | 跳转到指定行       |

---

### 建议学习路径（按优先级）：

1. **熟记终端的 Ctrl + A / Ctrl + E / Ctrl + R**（能显著提升命令行效率）
2. **掌握 Vim 的基本模式与保存/退出/复制粘贴操作**（因为 Vim 是 Linux 系统默认编辑器）
3. **Nano 适合应急快速改配置文件，快捷键也应掌握但可次之**
