在命令行终端（特别是 Bash / Linux shell 环境）中，除了 `Ctrl+C` 用于终止当前运行的前台进程，还有一系列常用的快捷键，建议你熟练掌握，尤其是在开发和调试嵌入式系统时非常实用。

---

## ✅ 常用快捷键分类总结如下：

### 一、控制进程相关

| 快捷键        | 功能说明                        |
| ---------- | --------------------------- |
| `Ctrl + C` | 强制终止当前前台运行的进程               |
| `Ctrl + Z` | 暂停（挂起）当前进程，送到后台             |
| `Ctrl + D` | 向终端发送 EOF，常用于退出 shell 或输入结束 |
| `Ctrl + \` | 强制退出并生成 core dump（SIGQUIT）  |
| `Ctrl + L` | 清屏，相当于 `clear` 命令           |

---

### 二、命令编辑（当前行编辑操作）

| 快捷键        | 功能说明                      |
| ---------- | ------------------------- |
| `Ctrl + A` | 光标移到行首                    |
| `Ctrl + E` | 光标移到行尾                    |
| `Ctrl + U` | 删除光标到行首的内容                |
| `Ctrl + K` | 删除光标到行尾的内容                |
| `Ctrl + W` | 删除光标前一个单词                 |
| `Alt + D`  | 删除光标后一个单词                 |
| `Ctrl + Y` | 粘贴上一次用 `Ctrl+U/W/K` 删除的内容 |
| `Ctrl + _` | 撤销上一个操作（某些终端有效）           |

---

### 三、命令历史操作

| 快捷键        | 功能说明           |
| ---------- | -------------- |
| `Ctrl + R` | 搜索历史命令（反向增量搜索） |
| `Ctrl + P` | 上一条命令（等同 ↑）    |
| `Ctrl + N` | 下一条命令（等同 ↓）    |

---

### 四、作业控制相关（结合后台任务）

| 快捷键 / 命令   | 功能说明                         |
| ---------- | ---------------------------- |
| `Ctrl + Z` | 暂停进程，变为后台挂起状态                |
| `bg`       | 将最近挂起的任务转为后台运行               |
| `fg`       | 将后台任务拉回前台运行                  |
| `jobs`     | 查看当前 shell 中的后台任务            |
| `kill %1`  | 杀掉第 1 个后台任务（由 `jobs` 列出的 %1） |

---

### 五、少见但实用

| 快捷键        | 功能说明               |
| ---------- | ------------------ |
| `Ctrl + S` | 暂停屏幕输出（软件流控，可能会卡住） |
| `Ctrl + Q` | 恢复被 `Ctrl+S` 暂停的输出 |

---

## ⚠️ 注意事项：

* `Ctrl + D`：在空行输入时表示退出当前 shell，不小心按到会关闭终端。
* `Ctrl + S` 和 `Ctrl + Q` 属于流控快捷键，有时在串口终端中会让输出卡住，不明白卡住原因时要试下 `Ctrl + Q`。
