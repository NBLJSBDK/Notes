---

## 1️⃣ 系统准备

确保你已经烧录了 `2025-10-01-raspios-bookworm-arm64-lite.img` 到 SD 卡，并能 SSH 或终端登录到树莓派。

先更新系统：

```bash
sudo apt update
sudo apt upgrade -y
sudo apt install sudo wget curl gnupg -y
```

---

## 2️⃣ 安装 X11 (X server) 和基本图形环境

安装最小的 X server：

```bash
sudo apt install xserver-xorg xinit x11-xserver-utils -y
```

* `xserver-xorg` → X11 核心
* `xinit` → `startx` 命令
* `x11-xserver-utils` → 辅助工具

---

## 3️⃣ 安装 i3 窗口管理器

```bash
sudo apt install i3-wm i3status suckless-tools -y
```

* `i3-wm` → 核心 i3
* `i3status` → 状态栏
* `suckless-tools` → 提供一些实用工具（例如 `dmenu_run`）

---

## 4️⃣ 安装 VSCode 和 Chromium

### VSCode (ARM64)

```bash
wget -O vscode-arm64.deb https://update.code.visualstudio.com/latest/arm64/deb/stable
sudo apt install ./vscode-arm64.deb -y
```

### Chromium

```bash
sudo apt install chromium -y
```

> 注意：在 Raspberry Pi 上，Chromium 会被系统默认提供为 `chromium-browser` 或 `chromium`，安装后可以通过命令 `chromium` 启动。

---

## 5️⃣ 配置 i3 快捷键启动应用

编辑 i3 配置文件：

```bash
mkdir -p ~/.config/i3
nano ~/.config/i3/config
```

在文件中添加（最小配置，只配置 F1/F2 和退出）：

```text
# 设置Mod键为Alt
set $mod Mod1

# F1 启动 VSCode
bindsym F1 exec code

# F2 启动 Chromium
bindsym F2 exec chromium

# 简单状态栏
bar {
    status_command i3status
}

# 退出 i3
bindsym $mod+Shift+e exit
```

保存后退出 (`Ctrl+O` → `Enter` → `Ctrl+X`)。

---

## 6️⃣ 启动图形环境

每次登录后运行：

```bash
startx
```

* i3 会启动
* 按 F1 → 打开 VSCode
* 按 F2 → 打开 Chromium

---

## 7️⃣ 可选优化（开机自动启动 i3）

编辑 `~/.bash_profile`（或 `~/.profile`）：

```bash
if [ -z "$DISPLAY" ] && [ "$(tty)" = "/dev/tty1" ]; then
    startx
fi
```

* 自动在第一个虚拟终端启动 X server + i3
* F1/F2 快捷键立即可用

## **一键脚本**
```bash
#!/bin/bash
# 一键配置 i3 + VSCode + Chromium (可中断恢复)
# 树莓派 5 / Raspberry Pi OS Lite (Bookworm ARM64)

LOG_FILE="$HOME/setup_i3.log"
mkdir -p "$(dirname "$LOG_FILE")"
exec > >(tee -a "$LOG_FILE") 2>&1

echo "=== 一键配置 i3 + VSCode + Chromium ==="
echo "日志文件: $LOG_FILE"
echo "请确保网络通畅"
read -p "按 Enter 开始..."

# 函数：安装 apt 包，如果失败记录但继续
safe_apt_install() {
    echo "--- 安装 $1 ---"
    sudo apt install -y "$1" || echo "安装 $1 失败，继续..."
}

# 更新系统
echo "--- 更新系统 ---"
sudo apt update && sudo apt upgrade -y || echo "系统更新出错，继续..."

# 安装 X11
safe_apt_install xserver-xorg
safe_apt_install xinit
safe_apt_install x11-xserver-utils

# 安装 i3 窗口管理器
safe_apt_install i3-wm
safe_apt_install i3status
safe_apt_install suckless-tools

# 安装 VSCode
echo "--- 安装 VSCode (ARM64) ---"
VSCODE_DEB="/tmp/vscode-arm64.deb"
if [ ! -f "$VSCODE_DEB" ]; then
    wget -O "$VSCODE_DEB" https://update.code.visualstudio.com/latest/arm64/deb/stable || echo "VSCode 下载失败"
fi
sudo apt install -y "$VSCODE_DEB" || echo "VSCode 安装失败"
rm -f "$VSCODE_DEB"

# 安装 Chromium
safe_apt_install chromium

# 配置 i3 快捷键
echo "--- 配置 i3 ---"
I3_CONFIG="$HOME/.config/i3/config"
mkdir -p "$(dirname "$I3_CONFIG")"
cat > "$I3_CONFIG" << 'EOF'
# 设置 Mod 键为 Alt
set $mod Mod1

# F1 启动 VSCode
bindsym F1 exec code

# F2 启动 Chromium
bindsym F2 exec chromium

# 简单状态栏
bar {
    status_command i3status
}

# 退出 i3
bindsym $mod+Shift+e exit
EOF

# 配置自动 startx 在 tty1 登录
echo "--- 配置自动 startx ---"
PROFILE_FILE="$HOME/.bash_profile"
LINE='if [ -z "$DISPLAY" ] && [ "$(tty)" = "/dev/tty1" ]; then startx; fi'
grep -qxF "$LINE" "$PROFILE_FILE" || echo "$LINE" >> "$PROFILE_FILE"

echo "=== 配置完成 ==="
echo "重启或重新登录 tty1 即可启动 i3"
echo "F1 打开 VSCode, F2 打开 Chromium"
echo "日志记录在 $LOG_FILE"

```
