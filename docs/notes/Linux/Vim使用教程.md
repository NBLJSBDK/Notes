# Vim 编辑器基础使用教程

Vim 是一个强大且高效的文本编辑器，常用于 Linux/Unix 系统下的开发与配置编辑。本文整理了 Vim 的常用命令和操作方法，适合初学者学习与参考。

---

## 一、Vim 启动方式

```bash
vim 文件名
```

* 如果文件存在，则打开编辑；
* 如果文件不存在，将创建一个新文件。

---

## 二、Vim 的三种常用模式

| 模式   | 进入方式       | 作用说明          |
| ---- | ---------- | ------------- |
| 普通模式 | 启动后默认进入    | 用于浏览、复制、删除等操作 |
| 插入模式 | 在普通模式按 `i` | 用于输入文本        |
| 命令模式 | 在普通模式按 `:` | 用于保存、退出、查找等命令 |

---

## 三、常用快捷键（按功能分类）

### 1. 模式切换（最关键）

| 快捷键   | 功能说明         |
| ----- | ------------ |
| `Esc` | 返回普通模式       |
| `i`   | 插入模式：从当前位置插入 |
| `a`   | 插入模式：从光标后插入  |
| `v`   | 可视模式（选中文本）   |
| `:`   | 进入命令模式       |

---

### 2. 文件操作命令（普通模式下）

| 命令    | 功能说明    |
| ----- | ------- |
| `:w`  | 保存文件    |
| `:q`  | 退出 Vim  |
| `:wq` | 保存并退出   |
| `:q!` | 强制退出不保存 |

---

### 3. 编辑命令（普通模式下）

| 命令         | 功能说明      |
| ---------- | --------- |
| `dd`       | 删除当前行     |
| `yy`       | 复制当前行     |
| `p`        | 粘贴内容      |
| `u`        | 撤销操作      |
| `Ctrl + r` | 重做（撤销的恢复） |

---

### 4. 光标移动（普通模式下）

| 快捷键   | 功能说明     |
| ----- | -------- |
| `h`   | 左移一个字符   |
| `l`   | 右移一个字符   |
| `j`   | 下移一行     |
| `k`   | 上移一行     |
| `0`   | 移动到行首    |
| `$`   | 移动到行尾    |
| `gg`  | 移动到文件顶部  |
| `G`   | 移动到文件底部  |
| `:数字` | 跳转到第 n 行 |

---

### 5. 查找与替换

| 命令          | 功能说明     |
| ----------- | -------- |
| `/关键词`      | 向下查找关键词  |
| `?关键词`      | 向上查找关键词  |
| `n`         | 查找下一个匹配项 |
| `N`         | 查找上一个匹配项 |
| `:%s/旧/新/g` | 全文替换     |

---

### 6. 插入模式操作

* 输入内容后按 `Esc` 退出插入模式，回到普通模式。

---

## 四、退出 Vim 的方式

| 命令    | 功能说明    |
| ----- | ------- |
| `:q`  | 退出（无修改） |
| `:wq` | 保存并退出   |
| `:q!` | 强制退出不保存 |

---

## 五、建议

* 熟练使用 `Esc`, `i`, `:wq`, `dd`, `yy`, `p`, `/关键词`, `n`, `G`, `gg` 这些命令即可满足大多数日常开发需求。
* 熟悉光标移动命令（`h`, `j`, `k`, `l`, `0`, `$`）能提高操作效率。
* 每天练习 5 分钟 Vim，是最快的进阶方式。

---

> Vim 的哲学是“**手不离键**”，你练得越熟，编辑效率越高。
