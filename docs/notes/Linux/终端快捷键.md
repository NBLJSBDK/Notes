# 终端快捷键（按重要程度排序）

| 快捷键        | 说明                                         |
| ---------- | ------------------------------------------ |
| `Ctrl + C` | **终止当前运行的程序**（你已掌握）                        |
| `Ctrl + L` | 清屏（等同于 `clear` 命令）                         |
| `Ctrl + A` | 光标移动到行首（**常用**）                            |
| `Ctrl + E` | 光标移动到行尾（**常用**）                            |
| `Ctrl + U` | 删除光标前整行（从光标到行首）                            |
| `Ctrl + K` | 删除光标后整行（从光标到行尾）                            |
| `Ctrl + W` | 删除光标前一个单词（**处理命令输入时很有用**）                  |
| `Alt + B`  | 向左移动一个单词（类似 shell 中的小跳转）                   |
| `Alt + F`  | 向右移动一个单词                                   |
| `Ctrl + D` | 发送 EOF（End of File），用于退出某些交互式程序或 `exit` 替代 |
| `Ctrl + Z` | 将当前进程挂起（后台运行，可用 `fg` 恢复）                   |
| `!!`       | 执行上一个命令（不算快捷键，但是历史命令中最常用的）                 |
| `Ctrl + R` | **反向搜索历史命令**（超级实用）                         |

---






在 **zsh** 中，有一套默认的键盘快捷键与行为，这些快捷键通常与 **bash** 中的快捷键相似，但也有一些额外的功能。以下是 **zsh** 默认的快捷键列表及其功能：

### 常用快捷键

| 快捷键        | 功能                          |
| ---------- | --------------------------- |
| `Ctrl + A` | 移动光标到行首（**常用**）             |
| `Ctrl + E` | 移动光标到行尾（**常用**）             |
| `Ctrl + U` | 删除光标前的内容（从光标到行首）            |
| `Ctrl + K` | 删除光标后的内容（从光标到行尾）            |
| `Ctrl + W` | 删除光标前的一个单词                  |
| `Ctrl + L` | 清屏（相当于 `clear` 命令）          |
| `Ctrl + R` | 反向搜索历史命令                    |
| `Ctrl + C` | 终止当前进程（**常用**）              |
| `Ctrl + Z` | 将当前进程挂起并移至后台（可以使用 `fg` 恢复）  |
| `Ctrl + D` | 发送 EOF（结束输入，通常退出 shell 或程序） |

### 移动光标的快捷键

| 快捷键        | 功能                           |
| ---------- | ---------------------------- |
| `Ctrl + A` | 移动光标到行首                      |
| `Ctrl + E` | 移动光标到行尾                      |
| `Alt + B`  | 向左移动一个单词（类似 `backward-word`） |
| `Alt + F`  | 向右移动一个单词（类似 `forward-word`）  |

### 编辑文本的快捷键

| 快捷键        | 功能                               |
| ---------- | -------------------------------- |
| `Ctrl + U` | 删除光标前的所有字符（从光标到行首）               |
| `Ctrl + K` | 删除光标后的所有字符（从光标到行尾）               |
| `Ctrl + W` | 删除光标前的一个单词                       |
| `Ctrl + Y` | 恢复最后一次删除的文本（粘贴）                  |
| `Ctrl + T` | 交换光标处的两个字符（类似 `transpose-chars`） |
| `Alt + D`  | 删除光标后的一个单词（类似 `kill-word`）       |

### 行编辑快捷键

| 快捷键                  | 功能                         |
| -------------------- | -------------------------- |
| `Ctrl + X, Ctrl + E` | 打开当前命令行到编辑器中（通常是 `vi` 编辑器） |
| `Ctrl + X, Ctrl + C` | 退出编辑器并返回到命令行（如果在编辑器中）      |

### 历史和文件补全

| 快捷键        | 功能                       |
| ---------- | ------------------------ |
| `Ctrl + R` | 反向搜索命令历史                 |
| `Ctrl + S` | 正向搜索命令历史                 |
| `Ctrl + P` | 查看上一个命令（类似 `up`）         |
| `Ctrl + N` | 查看下一个命令（类似 `down`）       |
| `Alt + .`  | 重复上一个命令的最后一个参数（**非常有用**） |

### 进程和作业控制

| 快捷键        | 功能              |
| ---------- | --------------- |
| `Ctrl + C` | 终止当前进程（**常用**）  |
| `Ctrl + Z` | 将当前进程挂起并放入后台    |
| `fg`       | 恢复并将最近挂起的进程放到前台 |
| `bg`       | 恢复并将最近挂起的进程放到后台 |

### 其他

| 快捷键                  | 功能                            |
| -------------------- | ----------------------------- |
| `Ctrl + L`           | 清屏，类似于 `clear` 命令             |
| `Ctrl + X, Ctrl + L` | 清除命令行内容（清空当前输入）               |
| `Alt + F`            | 向右跳跃一个单词（与 `Ctrl + Right` 相似） |
| `Alt + B`            | 向左跳跃一个单词（与 `Ctrl + Left` 相似）  |

### 自定义和扩展

你可以通过编辑 `~/.zshrc` 文件来修改 **zsh** 的快捷键绑定，使用 `bindkey` 命令来自定义每个快捷键的行为。常用的绑定方式如下：

```bash
bindkey "^U" backward-kill-line  # 修改 Ctrl+U 为删除当前行
```

### 总结

这些快捷键在 **zsh** 中帮助你更高效地操作命令行界面。你可以通过编辑 `~/.zshrc` 配置文件来自定义快捷键行为，以更好地适应个人习惯。如果你需要进一步定制或遇到问题，随时告诉我！
