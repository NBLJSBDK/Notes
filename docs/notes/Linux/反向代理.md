## 情况说明

你有：

* 本地机 SSH 端口：LOCAL_MACHINE_SSH_PORT
* Minecraft 端口：MINECRAFT_PORT
* Terraria 端口：TERRARIA_PORT
* ECS SSH 端口：ECS_SSH_PORT（公网可访问）
* ECS SSH 秘钥位置："ECS_SSH_KEY_LOCATION"
* ECS IP : ECS_IP_ADDRESS
目标：

* 通过 ECS 公网访问本地机的 SSH（LOCAL_MACHINE_SSH_PORT）、Minecraft（MINECRAFT_PORT）、Terraria（TERRARIA_PORT）。

---

## 1️⃣ ECS 端设置

1. 打开 GatewayPorts，让映射端口可以被公网访问：

```
sudo sed -i 's/^#\?GatewayPorts.*/GatewayPorts yes/' /etc/ssh/sshd_config
sudo systemctl restart sshd
```

2. 放行对应端口（防火墙或安全组）：

```
sudo ufw allow LOCAL_MACHINE_SSH_PORT/tcp
sudo ufw allow MINECRAFT_PORT/tcp
sudo ufw allow TERRARIA_PORT/tcp
```

---

## 2️⃣ SSH 反向隧道命令（本地机执行）

```
ssh -p ECS_SSH_PORT -N \
  -o ServerAliveInterval=60 -o ServerAliveCountMax=3 \
  -R 0.0.0.0:LOCAL_MACHINE_SSH_PORT:localhost:LOCAL_MACHINE_SSH_PORT \
  -R 0.0.0.0:MINECRAFT_PORT:localhost:MINECRAFT_PORT \
  -R 0.0.0.0:TERRARIA_PORT:localhost:TERRARIA_PORT \
  -i "ECS_SSH_KEY_LOCATION" note_user@ECS_IP_ADDRESS
``` 

说明：

* `-p ECS_SSH_PORT`：连接 ECS SSH 的端口
* `-N`：只做端口转发，不执行命令
* `-R 0.0.0.0:远程端口:本地主机:本地端口`：把本地端口映射到 ECS 公网端口，0.0.0.0 表示 ECS 上所有网卡的公网和内网都可以访问这个端口。
* `-R LOCAL_MACHINE_SSH_PORT:localhost:22`：ECS 上的 LOCAL_MACHINE_SSH_PORT 映射到家里 SSH
* `-M 0`：关闭监控端口
* `-f`：后台运行，如果用 systemd，就不需要 -f，systemd 会后台管理。
* 这里 ECS 上的端口：LOCAL_MACHINE_SSH_PORT（SSH）、MINECRAFT_PORT（Minecraft）、TERRARIA_PORT（Terraria）

> 注意：SSH 反向隧道只能转 TCP，如果 Terraria 用 UDP，这种方法就不行，需要用 WireGuard 或其它 UDP 转发工具。

## 3️⃣ 建议持久化（避免掉线）

用 `autossh` + systemd，让隧道自动重连。示例 service（本地机）：

~~~bash
sudo pacman -S autossh # Arch Linux
sudo nano /etc/systemd/system/reverse-tunnel.service
~~~

```
[Unit]
Description=Reverse SSH tunnel to ECS
Wants=network-online.target
After=network-online.target

[Service]
User=note_user
Environment="AUTOSSH_GATETIME=0"
ExecStart=/usr/bin/autossh -M 0 -N -o ServerAliveInterval=60 -o ServerAliveCountMax=3 -p ECS_SSH_PORT \
    -R 0.0.0.0:LOCAL_MACHINE_SSH_PORT:localhost:LOCAL_MACHINE_SSH_PORT -R 0.0.0.0:MINECRAFT_PORT:localhost:MINECRAFT_PORT -R 0.0.0.0:TERRARIA_PORT:localhost:TERRARIA_PORT \
    -i ECS_SSH_KEY_LOCATION note_user@ECS_IP_ADDRESS
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

启动和开机自启：

```
sudo systemctl daemon-reload
sudo systemctl enable --now reverse-tunnel.service
```

检查状态：

```
sudo systemctl status reverse-tunnel.service
```
如果显示 `active (running)`，说明隧道已经启动成功。

---

## 4️⃣ 如何访问

* SSH 本地机：`ssh -i 本地机秘钥 -p LOCAL_MACHINE_SSH_PORT 本地用户名@ECS_IP_ADDRESS`
* Minecraft：游戏里输入 `ECS_IP_ADDRESS:MINECRAFT_PORT`
* Terraria：游戏里输入 `ECS_IP_ADDRESS:TERRARIA_PORT`


---

## 5️⃣ 安全建议

* 本地机 SSH 必须用密钥登录，禁用密码登录
* ECS 防火墙尽量限制访问 IP
* 监控连接，必要时用 fail2ban 防止暴力破解
